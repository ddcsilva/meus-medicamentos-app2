# Shared Module

Componentes, diretivas, pipes e utilitários reutilizáveis compartilhados entre features.

## Estrutura

- **components/** - Componentes reutilizáveis (cards, botões, inputs, modais, etc.)
- **directives/** - Diretivas customizadas
- **pipes/** - Pipes de transformação de dados
- **ui/** - Componentes de UI específicos (botões, inputs, cards, etc.)
- **models/** - Interfaces, tipos e modelos compartilhados

## Regras

- Todos os componentes são standalone
- Componentes devem ser genéricos e reutilizáveis
- Models são interfaces TypeScript exportadas
- Pipes e directives são standalone
